# front-nuxt

## 環境構築

環境
`Mac M2`
`Docker version 24.0.6, build ed223bc`
`Docker Compose version v2.21.0-desktop.1`

### ビルド

``` sh
make build
```

インストールされるversion
`node v20.10.0`, `npm 10.2.3`, `npx 10.2.3`

### 起動

``` sh
make up
```

### 停止

``` sh
make stop
```
